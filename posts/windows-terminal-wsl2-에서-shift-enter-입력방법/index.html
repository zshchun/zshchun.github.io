Windows Terminal+WSL2 에서 shift-enter 입력방법

2022/05/31 · 87 words · Seunghwan Chun

Windows terminal 에서 shift-enter, shift-alt-enter 키입력을 위해서는 settings.json 에서 다음을 추가한다.

  • Windows terminal -> 설정 -> Json 파일 열기
1{ "command": {"action": "sendInput", "input": "\u001b[13;2u" }, "keys": "shift+enter" },
2{ "command": {"action": "sendInput", "input": "\u001b[13;5u" }, "keys": "ctrl+enter" },
3{ "command": {"action": "sendInput", "input": "\u001b[13;6u" }, "keys": "ctrl+shift+enter" },
4{ "command": {"action": "sendInput", "input": "\u001b\u001b[13;2u" }, "keys": "shift+alt+enter" },

참조#

Shift+Enter always submits, breaking PSReadline features
